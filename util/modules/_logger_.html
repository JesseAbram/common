# logger

▸ logger(_type: string): Logger

Defined in logger.ts:104

name: Logger

summary: Creates a consistent log interface for messages

description: Returns a Logger that has .log, .error, .warn and .debug (controlled with environment DEBUG=typeA,typeB) methods. Logging is done with a consistent prefix (type of logger, date) followed by the actual message using the underlying console.

example:

const l from '@polkadot/util/logger')('test');

l.log('blah'); // <date>     TEST: blah
